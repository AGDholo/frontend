# Contributing

When contributing to this repository, **please first discuss the change you wish to make** via
[issues](https://github.com/chafan-dev/frontend/issues)/[discussions](https://github.com/chafan-dev/frontend/discussions),
email, or any other method with the owners of this repository before making a change. 

Please note we have a [code of conduct](CODE_OF_CONDUCT.md), please follow it in all your interactions with the project.

## Pull Request Process

Please simply fork this repo and open a PR with description of the changes.

[README.md] contains instructions about how to work on this codebase locally.
